# AI Quant Research Platform

An AI-powered quantitative research platform for educational and research use.

The planned system will support A-share market scanning, technical signal
detection, AI-generated research reports, retrieval-augmented document search,
and a web dashboard.

## Status

Initial repository structure only. Business logic, scanner implementation, and
frontend implementation have not yet been started.

## Repository Layout

- `docs/`: Product, architecture, data, API, and roadmap documentation.
- `backend/`: Backend services and application APIs.
- `frontend/`: Web dashboard application.
- `scanner/`: Market scanning and technical signal detection.
- `ai/`: AI report generation and model integrations.
- `rag/`: Document ingestion, indexing, retrieval, and RAG orchestration.
- `data/`: Local data assets, schemas, fixtures, and data-processing workspace.
- `deployment/`: Deployment, infrastructure, and environment configuration.
- `tests/`: Cross-component and end-to-end test suites.
